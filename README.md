# Philomathia

Projet d’initiation aux bases mathématiques utiles pour la Data/IA.

## Objectifs
- Manipuler les notions clés (vecteurs, matrices, probabilités, indépendance, espérance, variance, écart-type, corrélation, statistiques descriptives, quartiles, boxplot, histogramme, TCL, dérivée).
- Savoir les expliquer simplement à un public non spécialiste.
- Les mettre en pratique dans un Notebook Python (Jupyter/Colab/Deepnote).

## Contenu du dépôt
- `philomathia.ipynb` — Notebook principal avec exercices, simulations et graphiques.
- `VEILLE.md` — Définitions rigoureuses + vulgarisation (images mentales/analogies).
- `slides/Philomathia.pptx` — Diapositives de synthèse pour la présentation.
- `README.md` — Contexte et guide rapide.

## Comment exécuter
1. Ouvrir le dossier dans VS Code (ou Jupyter Lab).
2. Lancer un kernel Python 3.x.
3. Ouvrir `philomathia.ipynb` et exécuter les cellules une par une (Shift+Enter).
4. Si un graphique ne s’affiche pas, vérifier que `matplotlib` est installé (`pip install matplotlib`).

## Pourquoi ces bases sont essentielles
Les mathématiques forment le langage des modèles: elles structurent les données, quantifient l’incertitude et permettent de généraliser. Sans ces briques, la programmation n’est que mécanique; avec elles, l’IA devient explicable, fiable et utile.
